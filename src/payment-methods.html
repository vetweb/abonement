Способы оплаты

Для удобства клиентов мы предусмотрели различные способы оплаты.

1. Банковский перевод

Получите по почте либо в офисе подписанный с нашей стороны Протокол согласования цены с реквизитами для оплаты. Укажите свои реквизиты и оплатите счёт.

2. Оплата безналичным путём через клиент-банк

Получите по почте либо в офисе подписанный с нашей стороны Протокол согласования цены с реквизитами для оплаты. Проведите платёж в интернет-банке, либо передайте бухгалтеру счёт для оплаты.

3. При помощи системы «Расчёт» (ЕРИП)

Оплатить услуги ООО «Артокс Лаб» вы можете через систему «Расчет» (ЕРИП), в любом месте, в комфортное время, в удобном для вас пункте банковского обслуживания — интернет-банке, с помощью мобильного банкинга, инфокиоске, кассе банков, банкомате и т.д.

Совершить оплату можно с использованием наличных денежных средств, электронных денег и банковских платежных карточек в пунктах банковского обслуживания банков, которые оказывают услуги по приему платежей, а также посредством инструментов дистанционного банковского обслуживания.

Для проведения платежа необходимо:
  1. Выбрать пункт «Система «Расчет» (ЕРИП).
  2. Выбрать интернет-магазины (сервисы).
  3. Выбрать «R».
  4. Выбрать Relax.by.
  5. Для оплаты «Услуги» ввести Номера счета (информацию о номере счета необходимо запросить у менеджера).
  6. Ввести сумму платежа (если не указана).
  7. Проверить корректность информации.
  8. Совершить платеж.

Если вы осуществляете платеж в кассе банка, пожалуйста, сообщите кассиру о необходимости проведения платежа через систему «Расчет» (ЕРИП).
